UML Diagrams
============

https://raw.githubusercontent.com/roshan-/auto/master/UML.png

This project uses UMLGraph to use text based UML Diagrams.
UMLGraph's Text based UML diagrams are
- easily editable
- version controlled
- easily compared
- intertwined into the code to be up-to-date
